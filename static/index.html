OpenAI Product Feed Validator

Validate a JSON, CSV, or TSV feed against common OpenAI product-feed rules. Upload a file or paste a public URL, then download a detailed report.

What this tool does

This validator checks your product feed for required fields (e.g., id, title, price, availability) and common formatting rules (like price format 129.99 USD and valid availability values). It supports JSON feeds (array of product objects) and CSV/TSV with a header row.

How to use

  1. Choose a file (.json, .csv, or .tsv).
  2. Select delimiter (for CSV/TSV) and encoding if needed.
  3. Click Start Validation to see errors & warnings, then download CSV/JSON reports.

Why OpenAI product feeds?

OpenAI’s commerce ecosystem uses structured product data to power AI-assisted discovery, comparison, and purchase flows. While it overlaps with Google Merchant Center feeds, there are differences in field names, accepted values, and format expectations. This tool focuses on OpenAI-aligned rules (e.g., availability enums and price format) and will evolve with the spec.

Note: This is an MVP validator. Always cross-reference the latest spec.
Tip: For large feeds, split into chunks. MVP hard cap: 50,000 rows per run.
Use a public, direct file URL. For GitHub/Gist, click Raw and paste that link.
